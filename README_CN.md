# skeleton-skill

> 让 Code Agent 轻松搭建专业项目架构的技能

## ✨ 特性

- 🧠 **架构级模板** — 面向可维护项目的生产级脚手架
- 💨 **零 Token 生成** — 从仓库拉取完整模板，而非逐行生成文件
- ⚡ **即时可用** — 一条脚本，数秒内得到完整项目布局
- 🔄 **持续更新** — 模板独立维护；新项目默认使用最新版本

## 📦 支持范围

TS/JS（CLI、React/Next.js、库）、Python（CLI、FastAPI、库）、Go、Rust。完整列表见 [templates](skill/references/templates.md)。

- 为 AI 智能体提供稳定、专业的项目脚手架
- 快速搭建可维护的项目结构
- 统一团队项目架构与规范
- 替代复制旧项目作为脚手架
